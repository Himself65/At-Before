# At-Before
UWP-倒计时应用</br>
# 介绍
此app会在 开始 菜单下显示某时刻的倒计时，记录重要的时刻，在电脑上时刻提醒你</br>



# 功能(正在开发中，基本完成的用 √ 表示)
* 磁贴显示倒计时 √
* 磁贴内容本地存储 √
* 添加背景
* 到达磁贴时间提醒
* 本地化（时区、显示方式、翻译）
